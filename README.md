# 🧠 AI Research Assistant Agent

An intelligent AI-powered research assistant built using **LangChain + LLMs** that can:
- Answer research queries
- Use external tools (search, Wikipedia, save)
- Generate structured research responses

---

## 🚀 Features

- 🔍 **Tool-augmented research**
  - Web search integration
  - Wikipedia lookup
  - Save results functionality

- 🤖 **LLM-powered reasoning**
  - Uses Claude (Anthropic) model
  - Can decide when to use tools

- 📄 **Structured output**
  - Responses are parsed into a clean format using Pydantic

---

## 🛠️ Tech Stack

- Python 🐍
- LangChain
- Anthropic (Claude model)
- Pydantic
- dotenv
